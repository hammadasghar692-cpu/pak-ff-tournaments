# 📂 Custom Images Folder

This directory contains the custom branding and high-fidelity screenshots for the **FF Tournament** web application.

You can customize the logos and phone mockup screenshots by uploading or overwriting the following files directly in this directory:

---

## 📸 File Specification Guide

### 1️⃣ Logo
* **Filename:** `logo.png`
* **Format:** PNG (transparent background recommended)
* **Dimensions:** 512x512 pixels
* **Usage:** App Header, Footer, Hero Badge, and loading screen.

### 2️⃣ Home Dashboard Screenshot
* **Filename:** `home.png`
* **Format:** PNG
* **Dimensions:** Vertical Phone Ratio (e.g., 1080x2400)
* **Usage:** Renders inside the interactive 3D phone mockup and screenshot galleries.

### 3️⃣ Tournament Lobby Screenshot
* **Filename:** `matches.png`
* **Format:** PNG
* **Dimensions:** Vertical Phone Ratio (e.g., 1080x2400)
* **Usage:** Displays as the tournament matches lobby slide.

### 4️⃣ Tournament Details Inset Screenshot
* **Filename:** `details.png`
* **Format:** PNG
* **Dimensions:** Vertical Phone Ratio (e.g., 1080x2400)
* **Usage:** Displays inside the mockups as the tournament specifications screen.

### 5️⃣ Community / Guild WhatsApp Screenshot
* **Filename:** `about.png`
* **Format:** PNG
* **Dimensions:** Vertical Phone Ratio (e.g., 1080x2400)
* **Usage:** Displays inside the mockups as the WhatsApp group connect screen.

---

## ⚙️ Development Note
The application uses fallback high-fidelity CSS-simulated mockups if you don't overwrite these files, so the app always looks professional out-of-the-box!
Once you upload your customized pictures, they will automatically override the default templates and display on the website!
